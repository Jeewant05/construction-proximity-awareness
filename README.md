# construction-proximity-awareness
Research and prototype notes for a cone-mounted proximity awareness system using wireless sensing and real-time alerts. 

This repository contains sanitized research notes, technology comparisons, and prototype planning material for a cone-mounted proximity awareness concept in construction environments.

The focus is on comparing wireless sensing options such as UWB, mmWave radar, RFID, BLE, LoRa/RSSI, Wi-Fi, GNSS-RTK, and vision-based sensing for real-time proximity alerts.

No sponsor-confidential documents, internal project files, field data, or proprietary implementation details are included.
